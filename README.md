# Imarticus LMS Assignment

Reusable MERN LMS with:

- Imarticus-style marketing landing page
- Course application and enrollment journey
- Admin-managed course/module/lesson/document content
- AI-powered document summarisation
- Optional Razorpay test-mode payment gate

## Apps

- `client`: React + Bootstrap frontend
- `server`: Node.js + Express + MongoDB API

## Planned Integrations

- MongoDB Atlas
- Cloudinary
- OpenAI or Gemini
- Razorpay (test mode)
